# Grid Proto

A tactical RPG prototype inspired by classic isometric strategy games.

**Play now:** https://grid-proto-taupe.vercel.app

## Features

- Isometric tile-based battle system with height and terrain
- Charge Time (CT) turn order system
- 20 job classes with unique abilities and progression
- Job prerequisite trees, JP earning, and ability learning
- Full damage formula system with elemental affinities and zodiac compatibility
- 24 status effects with cancellation and tick mechanics
- AI opponents with scored-action decision engine and behavior profiles
- 50+ equipment pieces across weapons, armor, and accessories
- 4-chapter campaign with 13 story battles
- World map navigation with procedural parchment-style art
- Dialogue and cutscene system
- Save/load system
- Sound effects and background music

## Controls

| Key | Action |
|-----|--------|
| WASD / Arrows | Pan camera |
| Q / E | Rotate map |
| Scroll | Zoom |
| Click | Select / Confirm |
| 1-3 | Menu shortcuts |
| Escape / 4 | Cancel / Back |
| M | Mute |
